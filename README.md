# Java Automation 8-Week Challenge

## 📌 О проекте
Учебный проект для изучения Java Automation (API тестирование).

## 🛠 Стек
- Java 17
- Maven
- JUnit 5
- RestAssured

## ✅ Реализованные тесты
- GET /posts
- GET /posts/{id}
- POST /posts
- PUT /posts/{id}
- DELETE /posts/{id}
- Фильтрация по userId
- Негативные сценарии (404, 400)

## 🚀 Запуск
```bash
mvn clean test
